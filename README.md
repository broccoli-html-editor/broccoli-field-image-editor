# broccoli-imageeditor-field

_broccoli-imageeditor-field_ は、 _broccoli-html-editor_ に カスタムフィールド "画像編集フィールド" を追加する拡張パッケージです。

## インストール - Install

```
$ npm install broccoli-html-editor --save
$ npm install broccoli-imageeditor-field --save
```
